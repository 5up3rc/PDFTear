# PDFTear
Malicious PDF document parsing tool

Author: [Cryin']

###Compatibility & Development: 

* Windows & Microsoft Visual C++ 6.0

###Features:

* 恶意PDF文档检测

* JavaScript解压缩

* 简单异或工具

* shellcode反汇编

* PDF关键字段数量提取
